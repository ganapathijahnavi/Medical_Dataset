# Thyroid Function Tests (TSH, T3, T4)

**Normal Range:**  
TSH: 0.4 - 4.0 mIU/L  
T3: 80 - 180 ng/dL  
T4: 4.5 - 12.5 mcg/dL

**Clinical Significance:**  
Assess thyroid gland function.

**Future Diseases:**  
- Hypothyroidism  
- Hyperthyroidism

**Lifestyle Recommendations:**  
- Follow thyroid treatment plans  
- Regular monitoring
